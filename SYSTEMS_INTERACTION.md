## 📊 Система взаимосвязей (обновлённая)

### **Диаграмма влияния**

```
                         ┌─────────────────────────────────────────┐
                         │      ЭКОНОМИЧЕСКАЯ СИСТЕМА             │
                         │  (Недельный доход, Расходы, Баланс)    │
                         └─────────────────────┬───────────────────┘
                                               │
              ┌────────────────┬───────────────┼───────────────┬─────────────────┐
              │                │               │               │                 │
              ▼                ▼               ▼               ▼                 ▼
   ┌──────────────┐  ┌──────────────┐ ┌──────────────┐ ┌──────────────┐ ┌──────────────┐
   │ КЛИЕНТЫ И    │  │ РЕПУТАЦИЯ И  │ │ ТОВАР И      │ │ ПОСТАВЩИКИ   │ │ СОТРУДНИКИ   │
   │ ПРОПУСКНАЯ   │  │ ЛОЯЛЬНОСТЬ   │ │ СКЛАД (FIFO) │ │ (надёжность) │ │ (команда)    │
   └──────┬───────┘  └──────┬───────┘ └──────┬───────┘ └──────┬───────┘ └──────┬───────┘
          │                 │                │                │                │
          │         ┌───────┴────────────────┴────────────────┴────────────────┘
          │         │
          ▼         ▼
   ┌─────────────────────────────────────────────────────────────────┐
   │           СОБЫТИЯ И РЕШЕНИЯ (35-50%/нед, 60-70% негативных)    │
   │  Конкурент, Проверки, Поломки, Поставщики, Персонал...        │
   └──────────────────────────┬──────────────────────────────────────┘
                              │
                              ▼
   ┌─────────────────────────────────────────────────────────────────┐
   │            СЕРВИСЫ КОНТУРА (7 шт, годовая оплата)              │
   │  Маркет, Банк, ОФД, Диадок, Фокус, Эльба, Экстерн            │
   └──────────────────────────┬──────────────────────────────────────┘
                              │
                              ▼
   ┌─────────────────────────────────────────────────────────────────┐
   │  РЕСУРС ВЛАДЕЛЬЦА (ЭНЕРГИЯ) + КАЧЕСТВО ЖИЗНИ                │
   │  Пассивный расход, Действия, Выгорание, Инвестиции в себя     │
   └──────────────────────────┬──────────────────────────────────────┘
                              │
                              ▼
   ┌─────────────────────────────────────────────────────────────────┐
   │  ДОСТИЖЕНИЯ, УРОВНИ, РЕКОРДЫ (бесконечная прогрессия)         │
   └─────────────────────────────────────────────────────────────────┘
```

**Ключевые отличия от v1.0:**
- Добавлены **Поставщики** и **Сотрудники** как самостоятельные системы
- **Энергия владельца** — центральный ограничитель, влияющий на все действия
- **События** — доминирующая система, создающая постоянное давление
- **Конкуренция** — отдельный трек, усиливающийся со временем

## 🔄 Основные циклы взаимодействия

### **Цикл 1: Доход → Расходы → Баланс → Действия (недельный)**

```
Недельный доход (25,000-35,000₽)
    │
    ├─→ Минус еженедельные расходы (5,600₽)
    │
    ├─→ Минус зарплаты и аренда (каждые 4 нед: 40,000-80,000₽)
    │
    ├─→ Минус подписки (годовые: списание при подключении/продлении)
    │
    └─→ Если баланс < 20,000₽ → АВАРИЙНОЕ СОСТОЯНИЕ
         │
         └─→ Нужно: сократить расходы ИЛИ увеличить доход
              │
              └─→ Но действия стоят энергию, а энергия может быть на исходе
                   │
                   └─→ ЛОВУШКА: нет денег на действия, нет энергии на заработок
```

**Пример (Магазин, неделя 4):**
```
Накоплено за 4 недели: ~28,000₽ (7,000₽/нед средний доход)
Еженедельные расходы: -5,600₽ × 4 = -22,400₽
Чистое накопление: ~5,600₽

Неделя 4: Платёж аренды + зарплата = -40,000₽
Баланс: 100,000 - 40,000 + 5,600 = 65,600₽

Если были улучшения или найм: баланс ещё ниже
Если событие конкурента на неделе 3-4: доход мог упасть на 15-20%

Вывод: Нужно копить с недели 1, не тратить на premières semaines
```

### **Цикл 2: Репутация → Клиенты → Чек → Прибыль → Репутация**

```
Высокая репутация (80+)
    │
    ├─→ ×1.1 множитель клиентов
    │
    ├─→ Больше потенциальных клиентов
    │
    ├─→ Если пропускная хватает → больше обслужено
    │
    ├─→ Больше выручка
    │
    └─→ Больше денег на сервисы и сотрудников
         │
         └─→ Сервисы/сотрудники дают пассивные бонусы
             │
             └─→ ещё выше репутация (положительный цикл)
```

**Обратный цикл (ловушка бедности v2.0 — усилен):**
```
Низкая репутация (< 20)
    │
    └─→ ×0.5 множитель клиентов
         │
         └─→ Мало клиентов → мало выручки
              │
              ├─→ Нет денег на сервисы (годовые — слишком дорого)
              │
              ├─→ Нет денег на сотрудников (зарплата каждые 4 нед)
              │
              ├─→ Конкуренция усиливает давление (-10-20% клиентов)
              │
              └─→ Негативные события подрывают и без того слабый бизнес
                   │
                   └─→ ЛОВУШКА БЕДНОСТИ (выйти сложнее, чем в v1.0)
```

**Как избежать:** Подключить Fokus (+1 реп/нед) или Market+OFD синергию (+1 реп/нед) как можно раньше.

### **Цикл 3: Энергия → Действия → Рост → Энергия**

```
Высокая энергия (80+)
    │
    ├─→ Можно сделать 2-3 действия за неделю
    │   (найм + улучшение + закупка = ~60 энергии)
    │
    ├─→ Быстрый рост: больше сотрудников, больше категорий
    │
    └─→ Рост бизнеса = больше пассивного расхода энергии (63-77/нед)
         │
         └─→ Энергия снижается быстрее!
              │
              └─→ Нужно инвестировать в качество жизни
                   │
                   └─→ Деньги вместо развития бизнеса
                        │
                        └─→ Замедление роста, но устойчивость
```

**Обратный цикл (выгорание):**
```
Низкая энергия (0-20)
    │
    ├─→ Можно сделать 0-1 действие за неделю
    │
    ├─→ Бизнес стагнирует
    │
    ├─→ Конкуренция усиливается, а игрок не может реагировать
    │
    ├─→ Бонусы сервисов снижены на 30%
    │
    ├─→ Если 2+ недели на 0: выгорание, -5 реп и -5 лояльн./нед
    │
    └─→ Если 6+ недель на 0: ПОРАЖЕНИЕ
```

**Как избежать:** Инвестиции в качество жизни (Gym, кресло, ноутбук) уменьшают пассивный расход. Сотрудники с "-N пассивного расхода/день" помогают. Семейный отдых (8,000₽) — экстренное восстановление.

### **Цикл 4: Поставщики → Товар → Качество → Клиенты**

```
Надёжный поставщик (качество 85%+)
    │
    ├─→ Хороший товар, клиенты довольны
    │
    ├─→ Нет негативных событий от поставки
    │
    ├─→ Стабильные цены
    │
    └─→ Можно открыть новые категории → больше клиентов
```

**Обратный цикл (плохой поставщик):**
```
Ненадёжный поставщик (качество <60%, надёжность <50%)
    │
    ├─→ Плохой товар → -10 репутации при поступлении
    │
    ├─→ Задержки → нехватка товара → потеря клиентов
    │
    ├─→ Исчезновение → потеря денег + срочный поиск
    │
    └─→ Контрафакт → -20 репутации, штрафы, клиенты уходят навсегда
```

**Как избежать:** Подключить Фокус (12,000₽/год) и проверять каждого поставщика перед сотрудничеством.

### **Цикл 5: Сервисы → Синергии → Эффекты → Стабильность**

```
Подключить Маркет (24,000₽/год, неделя 8)
    │
    ├─→ +8% пропускная, +8% чек, -15% потери товара
    │
    └─→ Доход растёт на ~12% (~3,000₽/нед)

Подключить ОФД (6,000₽/год, неделя 10)
    │
    ├─→ Синергия Market+OFD: +1 репутации/нед
    │
    └─→ Клиентов больше (через репутацию)

Подключить Фокус (12,000₽/год, неделя 14)
    │
    ├─→ +1 реп/нед + проверка поставщиков
    │
    └─→ Можно безопасно искать новых поставщиков

Подключить Эльба (18,000₽/год, неделя 18)
    │
    ├─→ +1 лояльности/нед, -25% энергии на операции
    │
    ├─→ Синергия Bank+Elba: +3% выручки
    │
    └─→ Экономия энергии = больше действий = быстрее рост

Итого: 4 сервиса = 60,000₽/год
Эффект: ~5,000-8,000₽/нед дополнительного дохода
Окупаемость: ~60,000 / 6,500 ≈ 9 недель
```

## ⚙️ Взаимодействие систем по пунктам

### **1. Экономическая система ↔ Сервисы Контура**

| Сервис | Цена/год | Эффект на доход | Эффект на расход | ROI (недель) |
|--------|----------|-----------------|------------------|--------------|
| **Market** | 24,000₽ | +12% (~3,000₽/нед) | 0 | ~8 |
| **Bank** | 18,000₽ | Косвенный (кредит) | -20% энергии | Косвенный |
| **OFD** | 6,000₽ | +1 реп/нед (косвенно) | 0 | Синергетический |
| **Diadoc** | 12,000₽ | +3% клиентов | -20% энергии закупки | ~15 |
| **Fokus** | 12,000₽ | +1 реп/нед + защита | 0 | Уникальный |
| **Elba** | 18,000₽ | +1 лояльн./нед | -25% энергии | ~12 |
| **Extern** | 24,000₽ | -2% налогов | 0 | 20+ (зависит) |

**Ключевое отличие от v1.0:** Годовая оплата делает ROI более долгосрочным. В v1.0 можно было "попробовать" сервис за месяц. В v2.0 — это годовая привязка, и решение должно быть взвешенным.

### **2. События ↔ Сервисы ↔ Сотрудники**

| Событие | Без сервиса/сотрудника | С сервисом | С сотрудником |
|---------|------------------------|------------|---------------|
| Налоговая проверка | -30,000₽, -5 реп | -5,000₽ (Extern) | — |
| Увольнение сотрудника | Потеря сотрудника | — | -5,000₽, остаётся (Elba) |
| Проблема с поставщиком | -8,000-20,000₽ | -2,000₽ (Market) | — |
| Пожар/затопление | -20,000-50,000₽ | — | -10% от стоимости (страх) |
| Рост цен поставщика | +10-20% цен | -5% от роста (Diadoc) | — |
| Жалоба клиента | -10,000₽, -8 реп | -3,000₽, -3 реп (Extern) | -5,000₽ (менеджер) |
| Сбой кассы | -3,000₽, -10% пропускн. | -1,000₽ (OFD) | — |

### **3. Поставщики ↔ Фокус ↔ Качество товара ↔ Клиенты**

```
Без Фокуса:
  Новый поставщик предлагает 30% скидку
  → Принимаем (не знаем рисков)
  → Поставщик ненадёжный (надёжность 40%)
  → Через 3 недели: партия некачественная
  → -10 репутации, -20% клиентов на 2 недели
  → Убыток: ~15,000₽ потерь + недополученный доход

С Фокусом:
  Тот же поставщик, 30% скидка
  → Фокус показывает: надёжность 40%, жалобы 12
  → Отказываемся или делаем тест-закупку
  → Ищем надёжного поставщика (надёжность 85%+)
  → Тест-закупка успешна
  → Стабильная поставка, довольные клиенты
  → Фокус окупил себя за 2 проверки
```

### **4. Энергия ↔ Сотрудники ↔ Качество жизни**

```
Без сотрудников и инвестиций:
  Пассивный расход: 49/нед (базовый)
  Остаток на действия: ~51
  → Можно: 1 найм (20) + 1 закупка (15) = 35, остаётся 16
  → Каждые 2-3 недели: накопленная усталость

С 2 сотрудниками (каждый -1 пасс./день):
  Пассивный расход: 49 - 14 = 35/нед
  Остаток: ~65
  → Можно: 1 найм (20) + 1 улучшение (25) + 1 закупка (15) = 60
  → Гораздо больше возможностей

С 2 сотрудниками + Gym (-1 пасс./день) + кресло (-1 пасс./день):
  Пассивный расход: 49 - 14 - 7 - 7 = 21/нед
  Остаток: ~79
  → Можно: 3 действия + ещё остаются резервы
  → Но: 5,000₽/4 нед (Gym) + 12,000₽ (кресло) = затраты
```

### **5. Реклама ↔ Уровень обслуживания ↔ Блогер**

```
Уровень обслуживания = (Репутация + Лояльность) / 2

Ситуация A: Репутация 70, Лояльность 65 → Обслуживание 67.5
  → Запускаем рекламу в соцсетях (18,000₽)
  → Блогер приезжает (40% шанс, неделя 3-4)
  → Обслуживание 67.5 >= 60 → Позитивное видео
  → +10 реп, +12% клиентов на 4 недели
  → Реклама окупилась!

Ситуация B: Репутация 45, Лояльность 50 → Обслуживание 47.5
  → Запускаем рекламу в соцсетях (18,000₽)
  → Блогер приезжает (40% шанс)
  → Обслуживание 47.5 < 60 → Разгромное видео
  → -15 реп, -20% клиентов на 4 недели
  → Плюс дополнительные потери на неделях 5-6
  → Реклама УБИЛА бизнес!

Вывод: Не запускайте рекламу при низком уровне обслуживания!
```

### **6. Категории товаров ↔ Поставщики ↔ Сотрудники ↔ Стадия**

```
Открытие новой категории требует:
  1. Поставщика с этой категорией (нужен надёжный)
  2. Сотрудника с нужной квалификацией (нужно нанять)
  3. Стадию развития, допускающую эту категорию (нужно достичь)

Пример: "Ресторанные блюда" для кафе
  → Нужен поставщик "СвежеПродукт" (надёжность 92%)
  → Нужен повар + официант (2 сотрудника)
  → Нужна стадия "Развивающийся" (неделя 21+)
  
  Если нет поставщика → нельзя открыть
  Если нет повара → нельзя готовить
  Если слишком рано (стадия "Стартап") → бизнес не готов

  Все три условия ОБЯЗАТЕЛЬНЫ
```

## 🎯 Баланс между системами

### **Золотой баланс (неделя 20, оптимизированный бизнес)**

```
Еженедельный доход:      +30,000₽ (с сервисами и сотрудниками)
Еженедельные расходы:    -8,000₽ (коммун., кассы, подписки качества жизни)
Еженедельная прибыль:    +22,000₽

Каждые 4 недели:
  Аренда: -25,000₽
  Зарплата (3 сотрудника): -19,000₽
  Итого: -44,000₽

Прибыль за 4 недели: 22,000 × 4 - 44,000 = +44,000₽

Параметры:
- Репутация: 60+ (растёт через Фокус и синергии)
- Лояльность: 65+ (растёт через Эльбу и менеджера)
- Баланс: ~120,000₽ (устойчивый рост)
- Энергия: ~60/100 (с сотрудниками и Gym)
- Команда: 3 сотрудника
- Категории: 3
```

### **Зоны опасности v2.0**

**Зона 1: Крах на неделе 4**
```
Признаки:
- Баланс < 55,000₽ к неделе 4
- Нет накоплений на аренду + зарплату
Решение: Минимизировать расходы на неделях 1-3, не инвестировать
```

**Зона 2: Ловушка бедности**
```
Признаки:
- Баланс < 30,000₽ на неделе 8+
- Нет сервисов, нет сотрудников
- Репутация падает, клиенты уходят
Решение: Срочно подключить дешёвый сервис (OFD 6,000₽/год)
```

**Зона 3: Выгорание владельца**
```
Признаки:
- Энергия < 20 на протяжении 3+ недель
- Нет инвестиций в качество жизни
- Все бонусы снижены на 30%
Решение: Срочно купить Gym (5,000₽/4 нед) + кресло (12,000₽)
```

**Зона 4: Катастрофа поставщика**
```
Признаки:
- Работа с ненадёжным поставщиком без Фокуса
- Некачественная партия → репутация падает
Решение: Подключить Фокус (12,000₽/год) до смены поставщика
```

**Зона 5: Крах от рекламы**
```
Признаки:
- Реклама запущена при обслуживании < 60
- Блогер разгромил бизнес
- Репутация < 30, клиенты уходят
Решение: Никогда не запускать рекламу при низком обслуживании
```

**Зона 6: Зарплатная ловушка**
```
Признаки:
- Слишком много сотрудников (зарплата >60,000₽/4 нед)
- Доход не покрывает зарплаты
- Сотрудники увольняются от неоплаты
Решение: Не нанимать больше, чем покрывает доход
```

## 📈 Траектория прогресса по неделям

### **Идеальная траектория**

```
НЕДЕЛЯ 1-4: Выживание
├─ Минимум действий, коплю деньги
├─ Баланс: 100k → 65k (после зарплатной недели)
├─ Реп: 50 → 48 (лёгкое снижение)
├─ Энергия: управляемо (51/100 остаток)
└─ Вывод: Нужно инвестировать

НЕДЕЛЯ 5-8: Первые шаги
├─ Подключаю OFD (6,000₽/год)
├─ Нанимаю 1 сотрудника (10,000₽ + 5,000₽/4 нед)
├─ Баланс: 65k → 55k (инвестиции)
├─ Реп: 48 → 49 (OFD помогает)
├─ Конкурент действует впервые (-15% клиентов)
└─ Вывод: Инвестиции начинают окупаться

НЕДЕЛЯ 9-12: Стабилизация
├─ Подключаю Фокус (12,000₽/год)
├─ Проверяю нового поставщика через Фокус
├─ Принимаю надёжного поставщика (качество 85%)
├─ Баланс: 55k → 70k (растёт!)
├─ Энергия становится проблемой: покупаю Gym (5,000₽/4 нед)
```

## 🎮 Механики как инструменты управления сложностью

### **Как каждая механика контролирует сложность:**

1. **Недельный цикл** — задаёт ритм, не позволяет действовать хаотично
2. **Энергия владельца** — ограничивает количество решений за неделю
3. **События (частые)** — не дают расслабиться, создают адаптивность
4. **Конкуренция** — растущий внешний давления, устраняет стагнацию
5. **Поставщики** — добавляют неопределённость в цепочку поставок
6. **Сотрудники** — создают управленческую нагрузку и расходы
7. **Улучшения (слабые)** — не позволяют решить проблему одним кликом
8. **Реклама (рискованная)** — наказывает за спешку
9. **Годовые подписки** — заставляют планировать на долгий срок
10. **Бесконечная игра** — устраняет "конечную паузу", сложности растут

### **Если игра слишком сложна (калибровка):**
- Увеличить стартовый капитал (100k → 120k)
- Снизить пассивный расход энергии (49 → 35/нед)
- Уменьшить частоту негативных событий (50% → 35%)
- Увеличить эффекты улучшений на 20%
- Снизить стоимость найма сотрудников на 30%

### **Если игра слишком проста (калибровка):**
- Увеличить частоту событий конкурента (каждые 2 недели вместо 3-5)
- Снизить эффекты синергий ещё на 20%
- Увеличить пассивный расход энергии при росте бизнеса
- Добавить больше типов негативных событий
- Уменьшить стартовый капитал (100k → 80k)
