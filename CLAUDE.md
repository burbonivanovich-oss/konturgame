# Бизнес с Контуром — План разработки

**Проект:** Экономический симулятор управления бизнесом (Business Simulator)  
**Стек:** React + TypeScript + Zustand + Tailwind CSS  
**Платформа:** Мобиль (портрет) + Веб (адаптив)

## 📝 Стандарты разработки

- **TypeScript strict mode** — полная типизация
- **Компоненты** — функциональные, с хуками, без классов
- **Состояние** — Zustand, минимумProps Drilling
- **Стили** — Tailwind CSS, BEM для сложных блоков
- **Коммиты** — на русском, описание и номер подзадачи (e.g., "1.1: Создать структуру папок")
