# Бизнес с Контуром — CLAUDE.md

Экономический симулятор управления малым бизнесом. Игрок выбирает тип бизнеса (магазин, кафе, салон красоты) и ведёт его бесконечно — до банкротства. Ключевая механика: 7 сервисов Контура решают реальные игровые проблемы и демонстрируют их ценность игроку.

**Стек:** React 18 + TypeScript + Vite + Zustand + Tailwind CSS

## Команды

```bash
npm run dev          # локальный сервер http://localhost:5173
npm run build        # production build
npm run type-check   # tsc --noEmit
npm test             # vitest
```

## Что реализовано

- **Игровой цикл:** `processWeek()` — 7 дневных итераций, счётчики недель/месяцев/лет
- **Сервисы:** 7 сервисов Контура + 7 синергий (`synergyEngine.ts`)
- **Достижения:** 20 достижений в 4 категориях, система уровней 1-10+
- **События:** 19+ событий с Контур-альтернативами + микрособытия
- **Реклама:** 10 кампаний, включая риск блогера, Campaign ROI аналитика
- **Улучшения:** 18 (6 на каждый тип бизнеса) в `UPGRADES_CONFIG`
- **Поставщики:** 6 штук (economy×2, standard×2, premium×2), влияние на цену и качество
- **Сотрудники:** 6 универсальных позиций (cashier, assistant, manager, specialist, supervisor, trainer)
- **Качество:** `qualityLevel` 0-100, влияет на репутацию, лояльность, средний чек
- **Pain Engine:** показывает потери от отсутствия каждого сервиса (стимул для покупки)
- **Онбординг:** прогрессивная разблокировка сервисов через `SERVICE_UNLOCK_MAP`
- **Промо-коды:** `PromoCodeModal`, `PromoWalletModal`, `BundleModal`
- **Микрособытия:** отдельный слой мелких событий (`MicroEventModal`)
- **Займы:** тип `Loan`, поле `loans: []` в стейте (v2.1)
- **Вехи:** `MilestoneView`, `milestoneStatus` (week10/20/30)
- **Кассы:** 3 типа (mobile/reliable/fast), `CashRegisterEngine`
- **Ассортимент:** категории товаров, `AssortmentModal`
- **Мобильный layout:** `MobileMainScreen` + `ResponsiveLayout`

## Что описано в документах, но не реализовано в коде

- **4-фазный недельный цикл** (Сводка → Действия → События → Результаты) — в UI нет принудительных фаз
- **Инвестиции в качество жизни владельца** (абонемент, ноутбук, кресло) — только в GAME_MECHANICS.md
- **Бизнес-специфичные сотрудники** — документы описывают поваров/барист/маникюрщиков, код использует 6 универсальных позиций
- **Волновая система достижений** — 4 волны только в GAME_MECHANICS.md, в коде плоский список
- **Стадии развития бизнеса** (Startup/Малый/Развивающийся/Средний/Крупный) — только в GAME_MECHANICS.md
