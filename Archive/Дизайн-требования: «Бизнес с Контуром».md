Дизайн-требования: «Бизнес с Контуром»
1. Шрифт
Семейство: Lab Grotesque K

2. Цвет
Общие правила
Фон: только белый или черный.

Акцентные цвета: оранжевый, зеленый, синий, фиолетовый.

Количество акцентных цветов на макете: минимум 3, максимум 4.

Один главный цвет, остальные — дополнительные.

Сочетание цветов при наложении
Контрастные пары: светлый + темный.

Гармоничные сочетания: оранжевый/зеленый (светлые) с синим/фиолетовым (темные).

❌ Запрещено: темный + темный, светлый + светлый.

3. Композиция и форма
Общие принципы
Простота, устойчивость. Форма без пустот внутри, на устойчивом основании.

Перекрытие фигур: не более 1/3 площади формы.

❌ Запрещено: глубокие впадины, вырваты, сложные вырубные формы, неустойчивые композиции с большими расстояниями между элементами.

Бенто (композиция без наложений)
Отступы: 1/8 базовой сетки.

Можно оставить место под логотип.

Допускается несколько фотомасок.

Наложение
Применяется, когда объектов больше 3 (в этом случае делаем бенто).

Перекрытие элементов: не более 1/5 площади друг друга.

Разрешённые типы наложения: форма + фотомаска + линейная графика.

❌ Сложные вырубные формы — не делать.

Скругления
Радиус внутреннего контейнера = радиус внешнего минус отступ между ними.

Радиусы и отступы — только числа, кратные 4:
4, 8, 12, 16, 20, 24, 32

4. Иконки
Количество и расположение
Использовать только по контексту.

Максимум 3 иконки в одном кейвизуале (KV).

Иконки должны стоять в линию.

С иконками можно использовать наложение, даже если объектов больше 3.

Запреты
❌ Иконки не накладываются на другие элементы.

❌ Иконки не накладываются на цветной фон.

Цветовые ограничения
Иконки вносят не более 2 дополнительных цветов.

Всего в KV может быть 4 цвета, половина из них — из палитры экосистемы (оранжевый, зеленый, синий, фиолетовый).

❌ Не использовать иконки, если они нарушают цветовую пропорцию.

5. Итоговые принципы стиля
Минимализм, структурность, контролируемые наложения, строгая цветовая и сеточная дисциплина, фирменный шрифт, аккуратное обращение с иконками.
